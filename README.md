# 🌸 Complete Digital Diary

A fully-featured, aesthetic Single Page Application (SPA) designed to help you track your daily musings, tasks, moods, and expenses. Built using pure HTML, modern CSS, and Vanilla JavaScript, this completely local diary ensures your data is saved and easily accessible.

## ✨ Features

- **📝 Daily Journaling**: Write daily notes, select your mood, and manage your to-do lists all in one place.
- **💰 Expense Tracker**: seamlessly log daily expenses and get real-time calculations for your daily expenditure.
- **📅 Monthly Summary**: A calendar-like view that compiles your log days, showing your selected mood and total expenses calculated over the month.
- **🎨 Dynamic Themes**: Customize the feel of your diary on the fly with a built-in interactive theme switcher. Includes:
  - Soft Pink (Default)
  - Dark Mode
  - Ocean Blue
  - Lavender
- **💾 Local Storage**: Your data persists securely in your own browser's Local Storage — no databases or server configurations required! 

## 🚀 Getting Started

No installation or build steps required! To start using the Complete Digital Diary:

1. Clone or download this repository.
2. Open the `index.html` file in your preferred modern web browser.
3. Start journaling, tracking expenses, and customizing your themes!

## 📁 Project Structure

- `index.html`: The main markup structure and application layout.
- `style.css`: Contains the design system, CSS variables for theme switching, and responsive design layouts.
- `script.js`: Handles data local storage operations, DOM manipulation, navigation routing, and logic.
